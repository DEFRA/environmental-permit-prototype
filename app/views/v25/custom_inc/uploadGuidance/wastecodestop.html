Download our template and use it to create your list of codes and descriptions (do not change the CSV file type).

You must use this format:

Code Description
01 04 09 waste sand and clays
06 01 04* phosphoric and phosphorous acid

Use the codes and descriptions from the List of Waste or European Waste Catalogue (EWC). You can find these in the waste classification technical guidance (opens new tab).

If a waste code ends in 99, you must write a full description of the waste. To restrict the type of waste you accept, add a note after the description..

Examples of restrictions

Add text after the description to:

  • restrict the quantity of the waste, for example: 16 06 01* lead batteries - no more than 500 tonnes
  • limit a particular waste within a code, for example: 16 01 03 - bicycle tyres only
  • exclude a hazardous property, for example: no corrosive waste
  • restrict a physical form, for example: 06 03 11* solid salts and solutions containing cyanides - no liquids
  • limit an attribute of the waste, for example: no odorous wastes
  • restrict specific substances or specific concentrations, for example: no chromium (VII) substances (greater than 1000 ppm)
  • describe container type or size limits, for example: no Intermediate Bulk Containers (IBCs)
